# Security Policy

## Reporting a Vulnerability

We take security seriously. If you discover a security vulnerability in Civilis, please **do not open a public issue**.

Instead, report it privately via email to:  
ðŸ“§ **security@civilis.ai**

Include:
- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

## Response Timeline

We aim to:
- Acknowledge your report within **48 hours**
- Provide a detailed response within **7 days**
- Release a patch within **30 days** (if applicable)

## Supported Versions

Only the latest stable release receives security updates.

## Public Disclosure

We coordinate disclosure with reporters and follow responsible disclosure practices.

Thank you for helping keep Civilis secure!
